```bash

# 2. Wait for services to start (watch for green checkmarks)
#    Takes about 2-3 minutes on first run
```

### Alternative: Local Development (Requires Node.js 18+)
```bash
# 1. Start backend services only
docker-compose up --build -d postgres zookeeper kafka ai-service

# 2. Start frontend locally (new terminal)
cd frontend
npm install    # First time only
npm run dev    # Starts on http://localhost:5173

# 3. Open browser
http://localhost:5173

# 4. Login with same credentials
```
